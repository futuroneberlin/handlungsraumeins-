# Räumliche Prinzipien

- Nähe bezeichnet semantische Verwandtschaft.
- Leere ist Denkraum, nicht Mangel.
- Überlagerung erzeugt theoretische Spannung.
- Drift macht Prozess sichtbar.
- Linien stellen soziale Relationen her.

Die Platzierung soll nie zufällig wirken. Jede Position ist eine Entscheidung über Abstand, Reibung und Lesbarkeit.
